JavaScript Variables



A variable is a “named storage” for data. We can use variables to store goodies, visitors, and other data.


There are 3 types of JS Variable:


  1. var
  2. let
  3. const

  1. JS Variable:- var

    var:- The var in JavaScript is used to to declare the variable in JavaScript globally inside the program.

    Let's see how to declare the variable:

    var message = 'Hello world'; /It is string datatype
    var age = 9; //It is number datatype


  2. JS Variable:- let


    let:- The let in JavaScript used to declare the the variable in JavaScript. It is access within the block. Outside the block it cannot be accessed.

    let message = 'Hello'; // It is string datatype
    let age = 9;// It is number.


  3. JS Variable:- const

    const:- The const in JavaScript is a container of data that can't be changed during the course of a program.

    const pi = 3.14;

    EX:- The value of pi is 3.14 and it's never gone changed. That's why we create the const as variable to us the value of pi.
